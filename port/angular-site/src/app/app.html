Reckless Drivin' — WebAssembly Port

A fan-made SDL2 / Emscripten port of the classic Mac OS 9 racing game Reckless Drivin' by Jonas Echterhoff. When clicking Start, hold the number key for the level you want to jump to in order to skip directly there. The original source code was released by the creator at github.com/jechter/RecklessDrivin . This effort builds on earlier ports such as Nathan Craddock’s work and Pomme by Jorio. View the GitHub repository for this port here
