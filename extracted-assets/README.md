# Extracted Assets

Level data and object type definitions extracted from `resources.dat` for *Reckless Drivin'* (Mac OS 9 game by Jonas Echterhoff).

Levels 1–3 are unencrypted; levels 4–10 use XOR encryption with the open-source registration key.

## Files

| File | Description |
|------|-------------|
| `levels.json` | Full level data for levels 1–3 (properties, objects, road segments) |
| `object-types.json` | Object type definitions (mass, dimensions, flags) for all types |
| `level-N-objects.csv` | Per-level object placement (typeRes, x, y, direction) |
| `level-N-road-segments.csv` | Per-level road geometry (leftX, centerX, rightX, segType) |
